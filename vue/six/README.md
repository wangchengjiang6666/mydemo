## vue-router

vue 路由 （前端路由）

根据 url 地址的不同，然后再页面上渲染不同的 vue 路由页面组件。
如果某个 vue 组件时跟着路由走的。那么这个 vue 组件我们称之为 vue 的路由页面组件。
